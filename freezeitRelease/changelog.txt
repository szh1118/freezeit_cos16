### v3.3.1SelfUse

- Fix foreground apps remaining `SIGSTOP`-suspended after a previous signal
  fallback freeze.
- Ensure every unfreeze sequence ends with `SIGCONT`.

### v3.3.0SelfUse

1. 后台 daemon 完成 Rust-only ARM64 收敛，活动代码树不再包含旧 C++ daemon、x64 fallback 或旧构建链。
2. 移除 F83 指纹、型号、内核与 readiness 硬编码；运行时报告区分验证目标、实际能力、控制许可和限制原因。
3. 修复 Binder freezer UAPI、PID 身份复验、signal 事务回滚、配置原子持久化和 fail-closed 控制门禁。
4. 将下载中延迟冻结、启动期异常解冻审计、Regular Refreeze 与大命令输出排空迁移到 Rust 并加入合同测试。
5. 保留 Xposed API 82/102 双栈；Athena hook 使用四套已审计 ROM 的 APK SHA、ROM fingerprint 和完整方法签名，未知组合 fail-closed。
6. 发布 ZIP 仅包含一个 ARM64 Rust ELF，并携带 GPL-3.0、对应源码说明、完整 SHA-256 和 provenance。
